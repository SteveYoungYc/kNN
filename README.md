# kNN
kNN implementation from scratch by C++

## Components
### CSV
Read and parse the CSV file
### KNN
Main logic of KNN
### Datapoint
Process one row of data into continuous features, categorical features and label

## Usage
On Linux
```shell
mkdir build
cd build
camke ..
make
./knn
```
